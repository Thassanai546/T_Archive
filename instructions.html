          --- Assembly Instructions ---
          

          Logical instructions:
        
            and:  performs a bitwise AND between two operands and stores the result in the destination operand.
            or:   performs a bitwise OR between two operands and stores the result in the destination operand.
            xor:  performs a bitwise XOR between two operands and stores the result in the destination operand.
            not:  performs a bitwise NOT on the operand.
            test: performs a bitwise AND between two operands but does not store the result.
      
          Arithmetic instructions:
      
            add: adds two operands and stores the result in the destination operand.
            sub: subtracts the second operand from the first operand and stores the result in the destination operand.
            mul: multiplies two operands and stores the result in the destination operand.
            div: divides the first operand by the second operand and stores the quotient in the destination operand.
            inc: increments the value of the operand by 1.
            dec: decrements the value of the operand by 1.
      
          Control flow instructions:
      
            jmp:      transfers control to the specified address.
            je, jz:   jumps if the zero flag is set (i.e., if the operands are equal).
            jne, jnz: jumps if the zero flag is not set (i.e., if the operands are not equal).
            jl, jng:  jumps if the sign flag is set (i.e., if the result of a signed operation is negative).
            jg, jnl:  jumps if the sign flag is not set (i.e., if the result of a signed operation is positive or zero).
            call:     calls a procedure at the specified address and pushes the return address onto the stack.
